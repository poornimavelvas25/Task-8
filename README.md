# Task-8
## Objective

The objective of this lab is to understand how a VPN works by installing a free VPN browser extension, verifying IP address changes, and analyzing its security features, benefits, and limitations.

## Tools Used

Proton VPN Browser Extension (Chrome)

Google Chrome Browser

IP checking website: https://whatismyipaddress.com

## Steps Performed

1️.Selected a Free VPN Extension

Visited the Chrome Web Store.

Chose a reputable VPN extension with good ratings.

Clicked Add to Chrome.

Installed the extension successfully.

2️.Signed Up / Logged In

Created a free account (if required).

Logged in through the extension popup.

3️.Connected to a VPN Server

Clicked on the VPN extension icon in the browser toolbar.

Selected a server location (closest or preferred country).

4️.Verified IP Address Change

Before connecting, visited: https://whatismyipaddress.com

→ Noted original IP address.

After connecting to VPN, refreshed the page.

Observed that the IP address changed to a different country/location.

This confirmed the VPN extension was working.

5️.Tested Browsing with VPN

Browsed different websites.

Confirmed HTTPS secure connection.

Verified that internet traffic was working normally.

## Benefits

Easy to install

Simple to use

Hides IP address

Protects browsing on public Wi-Fi

Good for bypassing geo-restrictions

## Limitations

Protects browser traffic only

Limited server locations in free version

Slower speed

May have data limits

Some free VPNs log data (must check privacy policy).

## Conclusion

A VPN browser extension was installed and tested successfully. The IP address change confirmed that the VPN was functioning correctly. While browser VPNs improve privacy during web browsing, they do not provide full system protection.

Clicked Connect.
